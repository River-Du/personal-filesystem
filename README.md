# Personal Filesystem

你有没有过这种经历：电脑桌面堆满文件，下载文件夹里几百个东西找不到，换了设备之后文件全乱套？

这是一套个人文件夹分类模板，帮你从源头建立一套统一的目录结构。适用于 PC、NAS、移动硬盘、U 盘、网盘等存储设备，复制过去就能用。

> 💡 这只是我的个人习惯，没有什么文件"必须放这里"、文件夹"必须这样命名"的规定。悉由尊便，好用就行！


## 文件夹结构


## 如何使用

克隆仓库或下载 ZIP 解压后，将整个目录结构复制到你的设备上（可以只选需要的部分），然后根据自己的习惯调整文件夹名称或增减子目录。

> `.gitkeep` 是为了让 Git 保留空目录而存在的占位文件。下载后如果不需要这些占位文件，可以手动删除或在项目根目录下运行以下命令删除：
>
> - Windows（CMD）：`for /r %f in (.gitkeep) do del "%f"`
> - Mac / Linux：`find . -name ".gitkeep" -type f -delete`


## 使用建议

- **编号前缀**：两位数编号确保文件夹按逻辑顺序排列
- **英文命名**：文件夹和文件建议统一使用英文
- **备份策略**：重要文件建议存放在多个设备或介质上

**Inbox（收件箱）**：所有来不及整理的新文件先放这里，比如刚下载的安装包、微信/邮件收到的附件、随手截的图、临时笔记。定期整理，有用的移走，没用的删除。

**Projects（项目）**：存放当前正在进行的事项，比如工作项目、正在写的论文、剪辑中的视频、本月的账单整理。建议按时间或事件建子文件夹整理。

**Personal（个人）**：个人证件、财务记录、简历、笔记等私密或重要的个人文件。`Vault` 可用于存放需要特别保管的内容。

**Resources（资源）**：从各处收集来的资料，如影视、音乐、书籍等，也包括需要保留在本地的资源文件。

**Media（媒体）**：自己拍摄的照片、录制的视频等个人媒体文件。建议按时间或事件建子文件夹整理。

**Software（软件）**：软件安装包、驱动程序、免安装便携工具等。方便重装系统或换设备时快速找回。

**More（更多）**：无法归入以上分类的内容，按实际需要自行扩展子文件夹。可按时间阶段或主题分类，如学校、工作、兴趣爱好、家庭等。

**Archive（归档）**：存放已结束的文件，比如已毕业的课程资料、已离职的前公司文件、往年旅游照片、旧手机备份等。建议按 `年份-事件名` 建子文件夹，如 `2025-毕业设计`.
